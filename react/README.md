# @lemy/react
Lemy Components for React

## Installation
```
npm install @lemy/react
```

## Usage

### Styles

with Sass:
```scss
@import '@lemy/react';
```

### Javascript Instantiation
```js
import React from 'react'
import { Button } from '@lemy/react'

class App extends React.Component {
  render () {
    return (
      <>
        <Button>Button</Button>
      </>
    )
  }
}

export default App
```
