# lemy-web-backend
Lemy Web Back-End

## Installation
```bash
$ cd /path/to/lemy-web-backend

# Install Node.js dependencies
$ npm install
```

## Available Scripts
In the project directory, you can run:

### `npm run cron`
Runs the cron.

### `npm start`
Runs the app.

### `npm test`
Runs the tests.
